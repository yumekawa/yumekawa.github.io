紹介
開発背景、環境、関連技術
開発背景:項目1：現在使っているの日本番組放送有料サイトはパソコンと移動設備が別々で払うしかないので 、
そのサイトの再生ソースをこのプロジェクトを通じて、自分のパソコンをサーバーとして移動設備で再生する
項目2：一部サイトの再生ソースやパソコン内のビデオは外で見えるため
開発環境:
OS:Microsoft Windows Version 22H2
SCI:I7-8700、GTX1060、16G-RAM
IDE:Eclipse Java EE IDE for Web Developers.Mars.2 Release4.5.2
Server:Apache-Tomcat-8.0.35
Browser:Mozilla Firefox-76.0-win64、Google Chrome-113.0.5672.126
Database:mysql-5.1.51-win32、HeidiSQL 8.3.4694
関連技術:JSP、JDBC、SQL、JavaBean、Servlet、Filter、JavaScript、CSS、XML、HTML
ビデオプレーヤー提供: https://github.com/DIYgod/DPlayer
↓
項目1
日本番組生放送と見逃しプロジェクト
●登録機能付き、利用するにはログインが必要（データベースに記録する）
●日本地上波民放放送見る可能（関東と関西各5社）
●限定時間内の番組見逃し可能（自分で時間を入力）
↓
項目2
オンラインビデオ管理システム
●登録機能付き、利用するにはログインが必要（データベースに記録する）
●ネット上のビデオリンクやパソコン内部の
  ビデオデータをウェブサイト内で再生する
●増加した内容管理できる（追加、削除、アップデート）
